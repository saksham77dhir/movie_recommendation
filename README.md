# 🎬 Movie Recommendation System

A Streamlit-based web application that recommends similar movies based on user input. The system uses a content-based filtering approach and enhances results with movie details like posters and plots using the OMDb API.

---

## 🚀 Features

* 🎥 Select a movie and get similar recommendations
* 🧠 Content-based filtering for accurate suggestions
* 🖼️ Displays movie posters
* 📖 Shows detailed movie plots
* ⚡ Fast and interactive UI built with Streamlit

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Scikit-learn
* OMDb API
* Requests

---
